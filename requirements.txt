# llm-feedback-control has ZERO third-party runtime dependencies.
# The core (the package in src/) uses only the Python standard library, so:
#
#     pip install llm-feedback-control
#
# pulls nothing else, and the deterministic pipeline runs immediately.
#
# To use the LLM path you need a model REACHABLE at runtime — this is NOT a pip
# dependency. Either:
#   * install Ollama (https://ollama.com) and pull a small model:
#         ollama pull phi3:mini
#   * or set CEILING_BACKEND=openai with OPENAI_API_KEY (stdlib HTTP, no SDK),
#   * or pass your own generate=... callable to any entry point.
#
# Optional extras (see pyproject.toml):
#   pip install "llm-feedback-control[aws]"   # boto3, for the EC2 tooling in aws/
#   pip install "llm-feedback-control[dev]"   # pytest, build, twine
